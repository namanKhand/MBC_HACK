# 3-Minute Demo Script

## Setup (Pre-demo)
- Wallet connected to Phantom
- Devnet SOL + USDC funded
- Example event pre-created

## Flow
### 1. Show the Problem (30s)
"Traditional ticketing has three major problems: scalpers drive 10x markups, no refunds for cancellations, and no reward for loyalty. EventGuard solves all three."

### 2. Create Protected Event (45s)
- Navigate to Organizer page
- Fill form: "Coachella 2026"
- Set price cap: 10% max markup
- Add Polymarket protection: "Weather" market, 50% refund if rain
- Click Create Event
- Show transaction confirmation

### 3. Buy Ticket (30s)
- Navigate to main page
- Click "Buy Ticket" on Coachella event
- Approve USDC transaction
- Show ticket in My Tickets

### 4. Demonstrate Rules (30s)
- Attempt resale at 25% markup
- Transaction fails with "PriceCapExceeded" error
- Explain: "Rules enforced on-chain, not just in UI"

### 5. Simulate Refund (30s)
- Show Polymarket market resolved to "Rain"
- Click "Claim Refund"
- Receive 50 USDC back
- Explain: "Automatic, no arguing with support"

### 6. Culture Passport (30s)
- Simulate check-in
- Show new Culture Badge minted
- Navigate to Passport page
- Show badge collection
- Explain: "Build reputation for VIP access"

### 7. Wrap Up (15s)
"EventGuard combines Solana's speed, Polymarket's prediction markets, and USDC for the future of ticketing. Thank you!"
